@mixin focused()

Applies focused styles to a link by adding an underline to the text.

Example

scss 
.element:focus-visible {
  @include focused;
}

Requires

@function rem()

$focus-width (number)

@mixin focused-box()

Applies focused styles to an boxed element by adding an outline and box-shadow.

Example

scss 
.element:focus-visible {
  @include focused-box;
}

Requires

@function rem()

$focus-text-color (Color)

$focus-background-color (Color)

$focus-width (number)
